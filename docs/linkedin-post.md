# LinkedIn launch post

I built **EzBoard** — a local-first collaborative canvas where people and AI agents can draw together in real time. ✨

What started as a lightweight whiteboard evolved into a full visual workspace for system architecture, brainstorming, wireframing, and AI-assisted diagramming.

The parts I am most excited about:

🔒 Boards stay in your browser — no account or hosted board database  
🤝 Encrypted peer-to-peer collaboration over WebRTC  
🧠 A provider-neutral AI protocol that works with local models, hosted models, CLI agents, and IDE agents  
🏗️ Built-in system-design components and 30 featured Excalidraw libraries  
⚡ Real-time cursors, presence, scene merging, exports, themes, and installable PWA support  
🌐 Deployed on GitHub Pages with a custom domain and enforced HTTPS

For the screenshot below, I asked an AI agent to draw a **30-layer zero-trust attendance system architecture**. It joined the same room as a peer and generated the diagram directly on the shared canvas.
